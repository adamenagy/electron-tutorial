Class: Markup

Markup

Base class for all Markups.
A Markup is a class that is capable of rendering itself as an Svg node.
It can also render itself into a canvas-2d context. Component within MarkupsCore extension.

Any class extending Markup should contain at least the following methods:

  • getEditMode()
  • set()
  • updateStyle()
  • setParent()
  • setRotation()
  • setSize()
  • setPosition()
  • renderToCanvas()
  • setMetadata()

A good reference is the rectangle markup implementation available in MarkupRectangle.js.
