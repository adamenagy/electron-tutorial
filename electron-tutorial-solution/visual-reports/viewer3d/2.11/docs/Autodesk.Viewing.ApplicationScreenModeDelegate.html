ApplicationScreenModeDelegate

Constructor

new ApplicationScreenModeDelegate(viewer)

ApplicationScreenModeDelegate class Allows viewer to go full screen. Unlike ViewerScreenModeDelegate class, this delegate doesn't use the full browser state, and it takes the entire page full screen, not just the viewer.
