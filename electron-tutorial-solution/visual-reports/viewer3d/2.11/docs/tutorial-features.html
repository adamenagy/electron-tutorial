Tutorial: Features

Features

Features come in 2 flavors, those accessed through the viewer instance and those accessed from extensions.

Viewer Features

  • Cut Planes: Cut geometry with invisible planes. Refer to setCutPlanes() and getCutPlanes().
    • Also check extension feature Section below for default UI controls.
  • Explode: Offsets nodes in a model which helps identify parts that make up a model. Refer to explode() and getExplodeScale().
  • Fit to view: Centers camera on the whole model, an individual node or a group of them.
  • Focal length: Get and set it through getFocalLength() and setFocalLength()
  • Full Screen: Refer to class ScreenModeDelegate.
  • LayerStates: Applies on 2d models only. Refer to getLayerStates() and activateLayerState().
  • ViewerStates: Save and Restore camera, selection and other values from LMV in Javascript Plain Objects.

Extension Features

  • Animation: Supports Fusion keyframe animation format, featuring camera and node transformations.
  • CAM: Computer Aided Manufacturing.
  • Collaboration: Share a viewing session with another user over the internet.
  • Comments: Provides Client side REST calls API into Autodesk's Commenting Service.
  • First Person: Overrides default camera navigation controls with First Person controls, common in videogames.
  • Gamepad: Provides navigation controls through a USB plugged game controller.
  • Markup: Annotate 2d and 3d models with arrows, text and custom defined shapes.
  • Measure: Support distance and angle measure for 2d and 3d models.
  • Oculus: Detects when an Oculus device is plugged and provides stereoscopic rendering
  • Simulation
  • Section: Provides UI controls over the viewer to alter planes that cut through the geometry
