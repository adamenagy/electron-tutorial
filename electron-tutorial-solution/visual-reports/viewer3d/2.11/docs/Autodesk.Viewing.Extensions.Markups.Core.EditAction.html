Class: EditAction

EditAction

Base class for all EditActions.
EditActions encapsulate Markup operations (such as creation, edition and deletion) that hook into the undo/redo system.

The minimum set of methods to implement on an EditAction extension are:

  • execute()
  • undo()
  • redo()

A good set of classes to check their implementation are:

  • CreateCircle.
  • DeleteCircle.
  • SetCircle.
